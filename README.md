# Web Scraping E-commerce Products

**Author:** Anil Haritwal 

Ye project **WebScraper.io** ke test e-commerce site se laptops, tablets, aur phones ke products scrape karta hai aur unhe CSV file me save karta hai.

---

## **Requirements**

- Python 3.x
- Libraries:
  - `requests`
  - `beautifulsoup4`
  - `csv` (built-in)

Install dependencies:

```bash
pip install requests beautifulsoup4
